# PMNSmain_MRI-Feb-2026
This includes R codes for PMNS MRI manuscript version dated 25 Feb 2026. 

# Analysis code for: [Revisiting the visceral fat paradigm: abdominal subcutaneous adiposity shows stronger cardiometabolic associations in young, lean Indians]

This repository contains the analysis code used in the manuscript:
Two separate scripts are present namely R_script_MRI_25Feb26 and All_study_comparison_plot.
R_script_MRI_25Feb26 includes main analysis performed in the manuscript
All_study_comparison_plot includes comparison of studies in published papers as mentioned in the manuscript

## Structure
- `R_script_MRI_25Feb26/` – R scripts for main analysis performed in the manuscript
- `All_study_comparison_plot/` – comparison of studies in published papers as mentioned in the manuscript


## Reproducibility
- R version: 4.4.1
- Key packages: tidyverse, broom, ggplot2

## How to run
1. Open RStudio
2. Set working directory to project root

Raw data are not shared due to ethical restrictions.
