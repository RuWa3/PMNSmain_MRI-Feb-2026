# PMNSmain_MRI-Feb-2026
This includes R codes for PMNS MRI manuscript version dated 25 Feb 2026. Two separate scripts are present namely R_script_MRI_25Feb26 and All_study_comparison_plot.
R_script_MRI_25Feb26 includes regression analysis performed in the manuscript
All_study_comparison_plot includes comparison of studies in publihed papers as mentioned in the manuscript
